# TaleMaker DeepSeek小说生成框架

## 项目简介
TaleMaker DeepSeek小说生成框架是一个基于DeepSeek API的小说生成工具，旨在为无经验用户提供便捷的小说创作解决方案。项目通过前端Vue界面实现参数化配置，用户只需输入世界观设定、人称视角、角色信息、情节要求等分类参数，即可快速生成高质量小说章节，并支持生成内容的实时预览、编辑、导出和导入功能。

## 功能特性
- **参数化配置**：提供世界观、人称视角、角色设定、情节要求等分类参数输入
- **实时生成预览**：流式传输技术实现内容实时生成与预览
- **多格式导出**：支持TXT/JSON格式导出，包含完整章节数据与思维链
- **历史管理**：本地存储生成历史，支持章节编辑与版本控制

## 安装与运行
### 前置要求
- 确认安装Node.js
- npm 6+ 或 yarn 1.22+
- 有效的DeepSeek API密钥

### 安装依赖
```bash
npm install
# 或
yarn install
```

### 运行项目
```bash
yarn run dev
```


## 使用指南

### 生成操作流程
1. 配置基础参数（世界观、视角等）
2. 添加角色信息与情节要求
3. 点击"生成小说章节"按钮
4. 实时预览生成内容
5. 编辑/导出/导入生成内容

### 导出格式说明
- **TXT格式**：包含章节内容与统计信息
- **JSON格式**：包含完整配置参数与生成历史

## 注意事项
1. **内容合规性**：禁止生成包含暴力、色情、歧视等违法内容
2. **API使用规范**：遵守DeepSeek API使用政策，避免高频请求
3. **数据安全**：API密钥不会存储在项目中，请妥善保管
4. **思维链功能**：仅支持DeepSeek Reasoner模型
5. **本地存储**：历史数据存储在浏览器本地，清除缓存将丢失数据

## 版本信息
- 框架版本：1.1.0
- 生成模型：deepseek-reasoner/deepseek-chat
- 技术栈：Vue 3 + OpenAI SDK + LocalStorage
